# Flying Junk

Casual Unity game prototype where the player controls a hole that swallows objects.

## Core Gameplay

- The player moves a hole around the scene.
- Objects appear and can be swallowed.
- Objects disappear on contact.
- Continuous eating builds a combo.
- Combo increases score.

## MVP

- One scene
- Controllable hole
- Few objects
- Objects disappear on contact
- Simple combo counter
- No extra systems

## Definition Of Done

- Hole moves
- Objects can be swallowed
- Objects disappear correctly
- Game runs without errors

## Scope Guardrails

Keep it simple and gameplay-focused.
Do not add extra systems outside MVP.
Do not describe environment setup.
